求24点
=================
####方法####
调用函数`solve24(n1,n2,n3,n4)`，n1,n2,n3,n4为四个正整数。
####示例####
求解：1，2，3，4
```
<!doctype html>
<html>
<head>
  <meta charset="UTF-8">
  <title>24Points</title>
</head>
<body>
  <h1>24Points</h1>
  <script src="solve24.js"></script>
  <script>document.write(solve24(1,2,3,4));</script>
</body>
</html>
```
得到答案`((1+2)+3)*4`。
